# Enterprise Knowledge Assistant

## Project Overview

This project demonstrates how to build an Enterprise Retrieval-Augmented Generation (RAG) AI Assistant from scratch.

The assistant will answer questions using company documents while running completely locally.

## Technologies

* Docker
* Ollama
* Phi-4
* Open WebUI
* Qdrant
* FastAPI
* Python
* Git
* GitHub

## Project Goals

* Learn AI Engineering
* Learn Docker
* Learn RAG
* Learn Vector Databases
* Learn FastAPI
* Build an enterprise portfolio project
